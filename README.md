# Kim2024_1-4xLPS
This repository contains the code and input data that was used for the analysis presented in the following publication:

Kim, J., Sullivan, O., Lee, K., Jao, J., Tamayo, J., Madany, A.M., Wong, B., Ashwood, P., Ciernia, A.V. (2024) [Repeated LPS induces training and tolerance of acute microglial responses across brain regions](https://link-springer-com.proxy1.library.jhu.edu/article/10.1186/s12974-024-03198-1). J Neuroinflammation. 2024 Sep 20;21(1):233. doi: 10.1186/s12974-024-03198-1. PMID: 39304952; PMCID: PMC11414187.

Images used in this study can be found on the Open Science Framework [repository](https://osf.io/gb7a9/).
